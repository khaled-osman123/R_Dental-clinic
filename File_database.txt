الملف مسؤول عن إنشاء قاعدة بيانات SQLite باسم clinic.db.

ينشئ 4 جداول رئيسية إذا لم تكن موجودة:

xrays → بيانات الأشعة

dental → العلاجات السنية

implant → الزرعات

press → الحشوات

كل جدول يحتوي على:

رقم تلقائي (Primary Key)

اسم المريض

العمر

الجنس

التاريخ

رقم التواصل

العنوان

وبعض الحقول الخاصة بكل نوع علاج

يستخدم:

sqlite3 للاتصال بقاعدة البيانات

CREATE TABLE IF NOT EXISTS لتجنب تكرار الإنشاء

commit() لحفظ التعديلات

استعلام لعرض الجداول للتأكد من إنشائها
